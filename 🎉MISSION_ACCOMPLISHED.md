## 🚀 立即使用

### 1. 安装

```bash
pnpm add @ldesign/code-editor monaco-editor
```

### 2. 快速开始

```typescript
import { createEnhancedCodeEditor } from '@ldesign/code-editor'

const editor = createEnhancedCodeEditor('#editor', {
  language: 'javascript',
  theme: 'tokyo-night',
  showLoading: true
})
```
